## Components

- os: **archlinux**
- package manager: **yay**
- display server: **xorg**
- compositor: **picom**
- windows manager: **dwm**
- bar: **dwmblocks**
- dmenu: **dmenu**
- terminal: **alacritty**
- editor: **neovim**
- wallpaper manager: **feh**
- audio server: **pulseaudio**
- file browser: **thunar**
- web browser: **firefox**
